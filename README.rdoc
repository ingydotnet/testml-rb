= testml-lite - Lite Version of TestML for Ruby

TestML is an Acmeist unit test language/framework. TestML::Lite is a
lighter, bootstrapping implementation of that.

= Synopsis

In a file called +test/mytest.rb+:

    require 'testml/lite'

    TestML::Test.new(
      :bridge => main.document = <<'.'
    Plan = 4;

    *x.add(*y) == *sum;
    *x.mult(*y) == *sum;

    === Test one
    --- x: 42
    --- y: 43
    .
